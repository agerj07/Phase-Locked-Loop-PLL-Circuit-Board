# Phase-Locked-Loop-PLL-Circuit-Board
Phase Locked Loop(PLL) Circuit Board utilized in multiphoton microscopy

Utilized KiCAD to design and construct a PLL circuit board for tissue imaging using Arduino as an interface to the AD9544 IC chip.
Reduced lab costs by 78%
